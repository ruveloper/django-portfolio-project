Python es un lenguaje de programación multipropósito y es uno de los lenguajes más usados en el mundo. Se utiliza en procesos de automatización, diseño de aplicaciones, machine learning, ciencia de datos, desarrollo backend, entre otros.

Django es el framework de desarrollo web-backend mas famoso de Python, con más de 15 años en el mercado, es ampliamente utilizado por empresas como Google, Instagram, Spotify y Mozilla que confían en él para construir aplicaciones web actualmente.

HTML, CSS y Javascript es el conjunto de tecnologías que cimientan la web, con ellas se estructura, se proporciona estilos y se otorga funcionalidad a cada página en internet.

Flutter es un framework de desarrollo multiplataforma creado sobre Dart, con él se pueden crear aplicaciones móviles, web o de escritorio desde una misma base de código. Fue desarrollado por Google en el 2011 y hasta el momento lo utilizan empresas como BMW, EBAY, Alibaba, Toyota y Nubank.
